# 💳 Banking Transaction Tracker

A multi-user banking transaction system built in Java using core Data Structures.  
The application supports account management, transaction history tracking, and secure balance validation through custom exception handling.

---

## 🚀 Features

- Create and manage multiple user accounts
- Deposit and withdraw money
- Maintain transaction history
- Undo last transaction using Stack
- Custom exception handling for insufficient balance
- Console-based interactive interface

---

## 🛠 Technologies Used

- Java
- Data Structures
- HashMap (Account management)
- Stack (Transaction rollback)
- Object-Oriented Programming
- Exception Handling
